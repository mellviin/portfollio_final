Adding additional packages unexpectedly caused build failures despite everything working during development.

## The investigation

- npm reported conflicting peer dependency versions.
- The production build was stricter than the development server.

## The fix

```diff
- Mixed package versions
+ Aligned React
+ Updated TypeScript
+ Clean dependency tree
```

Dependencies were upgraded together to maintain compatibility.

## Why it mattered

- Production builds became deterministic.
- Future package upgrades are easier to manage.
