Running the crawler across different systems exposed inconsistent dependencies and environment configurations.

## The investigation

- The application behaved differently between local machines.
- Python packages and runtime versions varied across environments.

## The fix

```diff
- Local machine setup
+ Dockerfile
+ requirements.txt
+ Containerized runtime
```

Every deployment now runs inside the same reproducible environment.

## Why it mattered

- Eliminated environment-specific bugs.
- Simplified onboarding and deployment.
