---

Search indexing failed whenever binary PDF files were processed using a plain text reader.

## The investigation

- Indexing crashed with UnicodeDecodeError on certain documents.
- The extraction pipeline lacked format-specific handlers.

## The fix

```diff
- open(file).read()
+ PdfReader()
+ python-docx
+ Text fallback
```

Each supported document type now follows its own extraction pipeline.

## Why it mattered

- Indexing became much more reliable.
- Users can search PDFs, DOCX files, and text documents seamlessly.
